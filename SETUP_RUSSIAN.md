# Запуск курса «Русский язык ЕГЭ 2026»

Курс встроен по той же схеме, что и информатика (платный контур уже настроен — см. `SETUP_PAYWALL.md`). Новых ключей ЮKassa не нужно. Нужно три шага: миграция → сид контента → передеплой функции оплаты. Затем `git push` (его делаешь ты — у песочницы нет твоих GitHub-кредов), Vercel соберёт фронт автоматически.

## Что уже сделано в коде
- Контент: `src/courses/russian/*.md` — 8 модулей, 35 уроков (+ итоги). Программа (витрина) — `m00_programma.md`.
- Метаданные курса: `src/courses/russian/index.js` (`RUSSIAN_COURSE`).
- Встроен в просмотр: `src/CourseStudy.jsx` (добавлен в `COURSE_META`, `getContentId` ловит «русск…»).
- Карточка в каталоге: `src/Platform.jsx` (`FALLBACK_CATALOG`, цена 4000 ₽, старая 5900 ₽).
- Сид-скрипт: `scripts/seed_russian.mjs`.
- Миграция каталога/доступа: `supabase/migrations/20260613000000_russian_course.sql`.
- Цена-фолбэк в оплате: `supabase/functions/create-payment/index.ts`.

## Шаг 1. Применить миграцию
Добавляет курс в каталог (`catalog_courses`, цена 4000 ₽, скидка с 5900 ₽, `is_published=true`) и выдаёт тебе как владельцу доступ.

```bash
supabase db push
```
(или вставь SQL из `supabase/migrations/20260613000000_russian_course.sql` в SQL Editor Supabase.)

## Шаг 2. Залить контент уроков в Supabase
Нужен **service_role**-ключ (Supabase → Settings → API). Не коммить его.

```bash
SUPABASE_SERVICE_ROLE_KEY="eyJ..." node scripts/seed_russian.mjs
```
Ожидаемый вывод: 8 модулей, ~43 записи (35 уроков + 8 итогов). Бесплатный урок — **1.1**. Скрипт идемпотентный: повторный запуск обновляет контент.

## Шаг 3. Передеплой функции оплаты
В `create-payment` добавлен фолбэк цены для `russian`. Цена всё равно берётся из БД (`catalog_courses`), но передеплой нужен, чтобы фолбэк совпадал.

```bash
supabase functions deploy create-payment
```

## Шаг 4. Запушить фронт
```bash
git add -A && git commit -m "Курс «Русский язык ЕГЭ 2026»" && git push
```
Vercel соберёт и выкатит сайт. После деплоя курс появится в каталоге, урок 1.1 откроется бесплатно, кнопка «Купить за 4 000 ₽» поведёт в ЮKassa.

## Проверка после деплоя
1. Каталог показывает 2 курса; у русского — бейдж «−32%» и зачёркнутые 5900 ₽.
2. Урок 1.1 открывается без оплаты; остальные — под замком (если ты не залогинен владельцем).
3. У тебя (maysbad@gmail.com) — полный доступ без оплаты.
4. Тестовый платёж в ЮKassa открывает курс целиком.

> Цену можно менять без передеплоя — правкой строки `russian` в таблице `catalog_courses` (поля `price` / `original_price`).
